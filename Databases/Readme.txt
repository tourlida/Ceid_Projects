1) Εγκαταστήστε την βάση δεδομένων League από το αρχείο databses_vol5.sql

2) Πρωτού τρέξετε το πρόγραμμα, ανοίξτε το αρχείο "LoadConnector.java" και εισάγετε στο όρισμα USER το όνομα της mysql σας πχ. root
   και στο PASSWORD τον κωδικό της ,πχ root123

3) Στο login στο πεδίο password εισάγετε το id του χρήστη και στο username το μικρο του όνομα. Το πρόγραμμα ελέγχει αν υπάρχει
 ο συγκεκριμένος χρήστης μ'αυτά τα στοιχεία username + password και μόνο τότε δίνει δικαίωμα πρόσβασης.

4) Υπάρχει περίπτωση για κάποιες κατηγορίες χρηστών όπως στο φίλαθλο, να μην εμφανίζει στον πίνακα των αγώνων μελλοντικούς ή προηγούμενους      αγώνες γιατί ανάλογα με την ημερομηνία μπορεί να μην έχουν παιχτεί για την εκάστοτε ομάδα. Γι'αυτό στην κλάση Controller στο αρχείο "Controller.java" υπάρχει μια μεταβλητή "today" όπου μπορείτε να ορίσετε μια ημ/νία ως 'σημερινή'. Οι ημ/νίες είναι από 2016-01-05 έως 2016-03-19.

ΥΓ* Σκεφτήκαμε μόλις ολοκληρώσαμε το πρότζεκτ να φτιάξουμε πεδία στα οποία ο χρήστης θα έβαζε τόσο τα στοιχεία της mysql (USER+PASSWORD),
όσο και να εισάγει μια ημ/νία ως 'σημερινή', όμως λόγω περιορισμένου χρόνου δεν το κάναμε.
